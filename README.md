# V2ray 订阅自动更新

自动爬取并更新V2ray/Karing/WinXray/Quantumult X订阅配置。

## 使用方法

### 方式1: 直接使用GitHub代理地址

在Karing中导入以下订阅地址：

```
https://raw.githubusercontent.com/YOUR_USERNAME/YOUR_REPO/main/v2ray_config.txt
```

或使用加速地址：

```
https://ghproxy.com/https://raw.githubusercontent.com/YOUR_USERNAME/YOUR_REPO/main/v2ray_config.txt
```

### 方式2: 使用GitHub Pages

1. 在仓库设置中启用GitHub Pages
2. 选择 `main` 分支作为源
3. 使用以下地址订阅：

```
https://YOUR_USERNAME.github.io/YOUR_REPO/v2ray_config.txt
```

## 更新频率

- 自动更新：每6小时
- 手动更新：在Actions页面手动触发
